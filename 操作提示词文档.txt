1. 项目概述
============================================================

这是一个通用的 Agent Skills 云端自动化工作台。

它不是单一的 AI 热点检索工具，而是一个可以被配置成任意信息监控任务的自动化系统：

- 配置 RSS、GitHub Search、第三方 API 等来源
- 按关键词和目标自动抓取候选信息
- 使用 OpenAI / Anthropic / 本地规则执行分析
- 生成结构化 JSON、HTML 报告和历史归档
- 本地用桌面 App 查看，云端用 GitHub Pages 查看

3. 如何配置一个新任务
============================================================

编辑文件：

config/sources.yml

重点字段：

workflow_name：任务名称
workflow_goal：任务目标
keywords：关键词列表
sources：信息来源列表

示例目标：

1. 创业项目日报
2. 竞品动态追踪
3. 论文与技术文章摘要
4. GitHub 项目观察
5. 投资机会雷达
6. 内容选题库

7. 常见问题
============================================================

Q：为什么不是 AI 分析？
A：OpenAI API 当前额度不足。开通 billing 或充值后会自动切换到 OpenAI 分析。

Q：能不能不做热点？
A：可以。修改 workflow_goal、keywords、sources，它就是任何主题的信息自动化工作台。

Q：为什么和 n8n 不一样？
A：n8n 是节点编排界面；这个项目把节点思想转换成代码配置 + GitHub Actions + 桌面 App。

8. 后续扩展提示词
============================================================

如果要继续让 Codex 扩展，可以使用：

请把这个 Agent Skills 自动化工作台扩展成「{你的任务名}」：
1. 修改 config/sources.yml 的 workflow_name、workflow_goal、keywords 和 sources
2. 调整 skills/hotspot-analyst/SKILL.md，使分析输出贴合该任务
3. 保留桌面 App 操作体验
4. 运行 python scripts/run_pipeline.py 验证
5. 提交并推送到 GitHub
