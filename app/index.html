Agent Skills 自动化工作台

现在可以做什么

  1. 配置任意主题关键词和信息来源
  2. 本地一键运行 Agent Skill 分析
  3. 查看信号卡片、Top 条目和来源状态
  4. 云端 GitHub Actions 定时自动运行
  5. GitHub Pages 自动展示最新结果

这个项目已经从单一热点页升级为通用 Agent Skills 自动化工作台。你可以把它改造成竞品监控、论文日报、GitHub 项目观察、内容选题库或任意信息雷达。
